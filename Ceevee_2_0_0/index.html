Ludo Board Game

A fully functional Ludo board game built with React.js and React Router. Players can select 2, 3, or 4 players, choose their favorite colors, and enter their names before starting the game. Once setup is complete, the game launches a responsive Main Board Screen where players roll dice, move tokens, and follow the classic Ludo rules. The gameplay includes features like safe zones, dice-based turns, and winning conditions, ensuring a fun, competitive, and engaging experience across devices.

  • Frontend: React.js
  • Routing: React Router
  • Interactive Game Logic
